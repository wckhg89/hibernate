# 하이버네이트 연습 프로젝트

'자바 ORM 표준 JPA 프로그래밍' 을 보고 연습을 하기 위한 프로젝트 입니다.

## 학습 목표

* EntityManagerFactory(SessionFactory), EntityManager(Session) 의 역할을 학습한다.
* JPA 영속성 컨텍스트가 어떻게 동작하는지 학습한다.
* 엔티티 매핑 및 연관관계 매핑에 대해 학습한다.

* (추가로) 깔끔한 REST API 구현을 연습한다.

## 프로젝트 구성

* JAVA 8
* Spring Boot 1.4.2
* Hibernate 4.3.4
* H2 DataBase
